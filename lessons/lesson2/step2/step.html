Mutable Variables

A variable in OCaml is immutable, you will never be able to change the value of x, its value is fixed:

x := x + 1


So to change the value of a variable, you can use mutable varaible:

let x = ref 42


You can then change the value of x

x := 56


! allows you to access the contents of the mutable variable:

let y = !x + 1
