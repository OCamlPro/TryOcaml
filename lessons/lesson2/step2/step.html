Mutable Variables

A variable in OCaml is immutable, you will never be able to change the value associated with x (you get an error):

x := x + 1


To get a traditional mutable variable, OCaml provides references:

let x = ref 42


You can then change the value of x

x := 56


The operator ! allows you to access the contents of a reference variable:

let y = !x + 1
