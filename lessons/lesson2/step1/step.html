Variables

If you need to use a value more than once, you can define a variable. To do that, we use the let syntax which allows for local variable declaration:

let x = 6 * 7


We can now check x in the toplevel:

x


And use it:

let y = x + 1


But a variable in OCaml is immutable, you will never be able to change the value of x, its value is fixed:

x := x + 1


That is a strengh of OCaml. So to change the value of a variable, you can use mutable varaible:

let x = 6 * 7


You can then change the value of x

x := 56


! allows you to access the contents of the mutable variable:

x := !x + 1
